Job Search Automation

A Python automation tool that searches multiple job platforms for QA leadership roles and generates reports highlighting new jobs since the previous run.

The project demonstrates API integration, data aggregation, automation, and CI/CD using GitHub Actions.

Features

Searches jobs using JSearch API

Expands leadership roles automatically

Supports multiple locations

Aggregates jobs from:

JSearch

Greenhouse boards

Lever boards

Removes duplicate job postings

Sorts jobs by posted date

Generates reports:

HTML

CSV

Highlights NEW jobs since the last run

GitHub Actions automation

Secure API key management via environment variables

Example Search Expansion

Searching for:

QA Manager

Automatically expands to:

QA Manager
Quality Engineering Manager
Test Manager
Automation Manager
SDET Manager
QE Manager
Example Command
python job_search.py --role "QA Manager" --location "Remote,Austin TX"
Command Line Arguments
Role (required)
--role "QA Manager"
Locations (optional)

Comma-separated list of locations.

Example:

--location "Remote,Austin TX,Dallas TX"

Default:

Remote
Generated Reports

Reports are generated in the /reports directory:

reports/
   jobs_YYYYMMDD_HHMMSS.html
   jobs_YYYYMMDD_HHMMSS.csv

The HTML report highlights new jobs since the previous run.

Environment Setup
1. Clone the Repository
git clone https://github.com/<your-username>/job-search-automation.git
cd job-search-automation
2. Create Virtual Environment
python3 -m venv .venv
source .venv/bin/activate
3. Install Dependencies
pip install -r requirements.txt
4. Configure Environment Variables

Create a .env file:

RAPIDAPI_KEY=your_api_key_here

Example file provided:

.env.example
GitHub Actions Automation

The repository includes a workflow that runs the job search automatically.

.github/workflows/job-search.yml

The workflow:

Runs on a schedule

Installs Python dependencies

Executes the job search script

Uploads reports as artifacts

Persists job history for comparison

Viewing Automated Reports

Reports generated by GitHub Actions are available under:

GitHub
→ Actions
→ Workflow Run
→ Artifacts

Artifacts include:

job-reports
job-history
