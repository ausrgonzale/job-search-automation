# AI Job Search Automation (Python)

A Python automation tool that searches for relevant QA leadership jobs and generates clean, readable reports.

This project demonstrates practical Python automation for real-world workflows and is intended as a portfolio project for QA Engineering Manager, Architect, and Principal level roles.

The script uses the **JSearch API** to retrieve job listings and automatically generates both **CSV** and **HTML reports** for easy review.

---

# Features

• Searches multiple locations for job opportunities  
• Expands searches for leadership roles (Manager, Director, Head of QA, etc.)  
• Uses the **JSearch API** for job discovery  
• Generates both **CSV** and **HTML** reports  
• Automatically opens the HTML report in your browser  
• Stores reports in a `/reports` directory for easy tracking  
• Easily customizable search keywords and locations

# Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/job-search-automation.git

cd job-search-automation

Install dependencies:

pip install -r requirements.txt


---

# Configuration

The script requires a **RapidAPI JSearch API key**.

Set your key inside the script:

API_KEY = "YOUR_RAPIDAPI_KEY"


You can obtain a key from:

https://rapidapi.com/letscrape-6bRBa3QguO5/api/jsearch/

---

# Running the Script

Run the job search automation:

python job_search.py


The script will:

1. Query the JSearch API
2. Expand searches across multiple roles and locations
3. Aggregate results
4. Generate reports
5. Automatically open the HTML report

---

# Example Use Case

This automation was created to streamline the daily job search process.

Instead of manually searching job boards, the script:

1. Pulls job listings from multiple locations
2. Filters leadership-level QA roles
3. Generates a readable report in seconds

This reduces job search time from **30+ minutes to under 10 seconds**.
